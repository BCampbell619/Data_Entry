The Wave Sheet

FOCUS:This guide will explain what the Wave Sheet is and what information it provides to the waver. The Wave Sheet is central to the coordination of work the waver handles.

A snap shot of what the Wave Sheet looks like:


The Wave Sheet contains a plethra of information. This guide will proceed with simple, straight–forward illustrations of each piece of information found on the Wave Sheet.

After reviewing the information given above you should understand the first dependency of the Wave Sheet; that is, a wave number has a route and a route has a shipping door. Remember a route is synonymous with a store, as explained in the Routing section.


Wave Sheet Boxes

Below is a depiction of the square boxes found next to the store column on the wave sheet.


*Hover over any of the 5 boxes to view their explanation.

00345

This first box is checked when the route has been activated.

This second box is checked when the ‘C, D, & Q’ tasks have been released for that wave.

This third box is checked when the wave tasks have been released.

This fourth box is checked when the wave has been picked, loaded, & the route has been closed.

This fifth box is checked when the wave has been invoiced, thus signifying the completion of the wave.

From left to right the boxes are marked in that succession – from activation to invoice and everything in between. Box 2 of the boxes does has a bit of detail that will be pointed out below.


Box 2 – C, D, & Q Tasks (Aisles QT & QU are excluded from this)

As the hover box describes, this box is marked when the C, D, & Q tasks have been released for the wave. However; for these modules – C, D, & Q – these tasks are released 10 waves ahead of the last opened wave. For example, if you have released through wave 44, then the eligible tasks for C, D, & Q will be released through wave 54. The wave sheet will then reflect this by having the second box checked through wave 54.

Add Boxes

In columns H, P, X, AF, and AN the 5 boxes will need to be entered.


Highlight each of the columns starting beneath the “Round” row and delete any text within those columns.


Next change the font to ‘Wingdings’ for each of those columns.


Once the font has been changed enter 5 lowercase q's into one cell of any one of those columns.


Copy and paste the q's into all the remaining cells for columns H, P, X, AF, and AN.

Set Font Sizes

There are two font sizes for the entire sheet that need to be set – one for the boxes (q's) and one for all the other text.


The boxes need to be set to a font size of 32.


The door numbers, wave numbers, and the store numbers need to be a font size of 24.
