How to Create a Route

FOCUS:This guide will take you step by step through the process of creating a route for a new store.


Select the menu option titled ‘Route Maintenance – rtl’ and the screen below will appear.

When the screen shown below is reached you can enter any valid route number, or leave it blank, and press ‘Enter’.


Once a route (i.e. the store number) is entered (or not) all routes will be shown in sequence. Use option ‘3’ to copy any of the displayed routes and press ‘Enter’.

Option 3 is used to copy:


The next screen will prompt for the new route number. Enter the new route number in the ‘Route Number’ field and press ‘Enter’.

Depiction of a new route entered:


The next screen will have a lot of information to display. Nothing needs to be done here. Press ‘Enter’ and ‘F16’ to create the route.

Route creation detail information screen:


Once the route has been created it will be displayed where ever it falls within the numberical sequence.


Page down or press ‘F12’ to back up to the first screen and enter the new route number to see if it has been created.


Once this is acheived you have successfully created a route. This new route is now eligible to be activated and attached to a shipping dock door.
